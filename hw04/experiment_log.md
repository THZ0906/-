## 1. 实验环境
- 操作系统：Windows 11
- Python 版本：3.1.4
- 硬件：CPU:i7，GPU：RTX3050
- 依赖安装方式：pip

## 2. 安装过程
执行命令如下：
python -m pip install --upgrade pip
python -m pip install openai-whisper
python -m pip install torch torchvision torchaudio
